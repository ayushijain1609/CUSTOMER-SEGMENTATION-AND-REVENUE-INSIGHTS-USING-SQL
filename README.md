# CUSTOMER-SEGMENTATION-AND-REVENUE-INSIGHTS-USING-SQL
A SQL-based data analysis project that segments customers based on purchase behavior and spending patterns.

Objective: To identify high-value customers and segment them for targeted marketing.
Dataset: Customer transactions (orders, amount spent, frequency).
SQL Queries Used:
      Aggregations (SUM, COUNT, AVG).
      Joins for merging customer & order data.
      Window functions for ranking.

Tools: MySQL
Result: Classified customers into segments such as loyal, churn-risk, and high spenders.
